## 주요 기능

- 시네마틱 다크 브라운 톤 디자인
- GSAP + ScrollTrigger 스크롤 애니메이션
- 데스크톱 / 모바일 반응형
- 필모그래피 (영화, 드라마, 연극/뮤지컬)
- 갤러리 + 라이트박스
- 연기영상 / 출연영상 바로가기

## 기술 스택
